# HappyMe
## App Identity

**App Name:** HappyMe 
**Tagline:** Handmade home decorations made with love

**Description:**  
HappyMe Twiin is a handmade home decoration brand that creates unique,
artistic pieces for modern homes. This project transforms the Instagram
business into a complete digital platform (mobile apps + website).
